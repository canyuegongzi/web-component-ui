## web-component-ui

### 准备

1. 执行 ```yarn```；
2. 执行 ```lerna bootstrap```。

### 开发

1. 进入 packages/web-ui；
2. 执行 ```npm run start:dev```，实时编译组件库；
3. 进入 example 目录；
4. 执行 ```npm run webpack:dev```, 进行组件实时预览。

### 构建

1. 进入 packages/web-ui；
2. 执行 ```npm run build:tsc```，构建打包； ；
3. 执行 ```npm run webpack:dev```, 进行组件实时预览。
